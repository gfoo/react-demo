## Local dev

Create a .env file contaning dev configurations (required backend [here](https://github.com/gfoo/fastapi-demo)) :

```
REACT_APP_API_URL="http://localhost:3000"
```

Install dependencies and launch:

```shell
$ npm install
$ npm start
```
